# Bill of materials

## Main parts

| Part | Version / type | Notes |
|---|---|---|
| Computer | Raspberry Pi Zero 2 WH | WH means GPIO header is already soldered |
| Battery | PiSugar 3 1200mAh Raspberry Pi Zero Battery | Clean portable power |
| Displays | 2x 0.96 inch SSD1306 I2C OLED, white | Used as eyes |
| Microphone | INMP441 I2S MEMS mic | Round MH-ET LIVE style module |
| Amplifier | MAX98357A I2S 3W Class D amp | Drives the speaker |
| Speaker | 3525 4 ohm 3W cavity speaker | 25 x 35 mm body |
| Button | 16 mm momentary illuminated metal button | Single color LED is easiest |
| Wires | 20 cm female-to-female Dupont kit | Can be split into loose wires |
| Screws | M3 x 8 self-tapping | For the printed case, not for Pi mounting |

## Notes

- Use M3 screws for the case.
- The Pi and PiSugar usually use their own small hardware.
- If the speaker has a small connector, cut it off, strip the wires, and put red/black into the MAX98357A screw terminal.
