# Raspberry Pi setup notes

Recommended OS:

- Raspberry Pi OS Lite
- no desktop

Goal:

- boot headless
- connect to WiFi
- run Mimo automatically
- show simple boot/status on OLED
- shut down after idle time to save battery

Estimated boot time:

- normal: 25 to 45 seconds
- optimized: 15 to 30 seconds

Power-saving ideas:

- turn OLEDs off when idle
- disable HDMI
- disable Bluetooth if unused
- keep speaker amp disabled until speaking
- auto shutdown after 10 to 15 minutes idle
